# Wine Classification — Machine Learning From Scratch

A Machine Learning course assignment that implements 8 classic ML algorithms **from scratch (NumPy only, plus a PyTorch MLP)** and benchmarks them on the [UCI Wine dataset](https://archive.ics.uci.edu/dataset/109/wine): 178 samples, 13 features, 3 classes.

## Methods implemented

1. Principal Component Analysis (PCA) — for 2D visualization
2. Least Squares classifier
3. Logistic Regression with gradient descent
4. K-Nearest Neighbors (KNN)
5. Gaussian Naive Bayes
6. Multilayer Perceptron (PyTorch)
7. Support Vector Machine (one-vs-rest, hinge loss)
8. K-Means clustering

## Getting started

```bash
pip install -r requirements.txt
jupyter notebook wine_classification.ipynb
```
